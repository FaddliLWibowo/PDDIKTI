Feeder PDDIKTI PDDIKTI (Pangkalan Data Pendidikan Tinggi) adalah sebuah pusat kumpulan data penyelenggara pendidikan tinggi seluruh Indonesia. Kumpulan data tersebut dikelola oleh Dirjen Dikti yang beralamatkan di http://forlap.dikti.go.id/ yang merupakan data hasil sinkronisasi aplikasi PDDIKTI yang dikelola oleh masing-masing perguruan tinggi nasional.

PDDIKTI merupakan kelanjutan dari program EPSBED layar biru berpaltform MS-DOS yang dikembangkan menjadi aplikasi berbasis web yang dijalankan secara lokal dan bisa disinkronisasikan datanya apabila telah terjadi update data baik data lokal PT maupun data di server FORLAP. Sebelum adanya Surat Edaran Dirjen Dikti No. 2332/E1.2/KP/2015 tanggal 28 April 2015 tentang Feeder PDDIKTI laporan penyelenggaraan PT dilaksanakan dengan FTP yang dibantu oleh Kopertis. Setelah terbitnya SE tersebut PT (khususnya PTS) bisa langsung mengirim datanya ke Dikti melalui proses sinkronisasi PDDIKTI.

Aplikasi PDDIKTI Aplikasi PDDIKTI bisa diunduh di halaman ini lengkap beserta dengan buku petunjuk penggunaan dan patch aplikasinya. Sebelum bisa menggunakan aplikasi tersebut, masing-masing operator PDDIKTI harus memiliki data prefill dan kode hasil generate dari halaman FORLAP. Oleh karena itu, operator harus mengetahui user name dan password halaman FORLAP tersebut untuk bisa memasang dan menggunakan aplikasi PDDIKTI. Bahkan user name dan password PDDIKTI tersebut tersinkronisasi dengan user name dan password pada halaman FORLAP. Sehingga apabila kita mengganti password pada halaman FORLAP dan melakukan sinkronisasi setelahnya, maka password pada aplikasi PDDIKTI otomatis ikut terbarui.

Instalasi Proses instalasi dan pengisian data prefill sudah dijelaskan lengkap pada buku Panduan Aplikasi Feeder, silahkan dibaca dan diikuti sesuai langkah yang dijelaskan.
